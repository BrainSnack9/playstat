# PlayStat 서비스 가이드라인

## 서비스 정의

PlayStat은 **합법적인 스포츠 데이터 인사이트 플랫폼**입니다.

- 경기 데이터 기반
- AI가 자동 분석 리포트를 생성
- SEO 최적화 및 애드센스 승인 가능한 합법적 콘텐츠

## 핵심 원칙

### 제공하는 것 (O)

- 경기 데이터 분석
- 팀/선수 통계
- AI 자동 분석 리포트
- 최근 폼 분석
- 상대전적 (H2H)
- 홈/원정 성향 분석
- 객관적인 데이터 인사이트

### 절대 제공하지 않는 것 (X)

- 승부 예측 ("A팀이 이길 것이다")
- 배팅 추천 ("A팀에 배팅하세요")
- 배당률 분석
- 도박 조장 콘텐츠
- 불법 도박 사이트 연결

## 콘텐츠 작성 규칙

### AI 분석 리포트 작성 시

**허용되는 표현:**
- "A팀은 최근 5경기에서 4승 1무를 기록하며 좋은 폼을 유지하고 있습니다"
- "B팀은 원정에서 평균 1.2골을 실점하고 있습니다"
- "양 팀의 최근 5번의 맞대결에서 A팀이 3승을 거뒀습니다"
- "홈 팀의 평균 득점은 2.1골입니다"

**금지되는 표현:**
- "A팀이 이길 것으로 예상됩니다"
- "B팀에 배팅하는 것이 좋습니다"
- "승률 70%로 A팀 승리가 유력합니다"
- "오버/언더 2.5골 추천"

### SEO 콘텐츠 작성 시

- 객관적인 데이터 중심의 제목 사용
- 예측/추천 키워드 사용 금지
- 분석/인사이트/통계 키워드 권장

## 비즈니스 모델

1. **오가닉 트래픽**: SEO 최적화된 분석 콘텐츠
2. **애드센스**: 합법적 콘텐츠로 승인 가능
3. **타겟 유저**: 스포츠 분석에 관심 있는 사용자 (토토 유저 포함, 하지만 배팅 추천은 절대 금지)

## 법적 준수 사항

- 불법 도박 조장 금지
- 스포츠토토(베트맨) 외 도박 사이트 언급 금지
- 승부 예측 서비스로 오해받지 않도록 주의
- 데이터 인사이트만 제공, 판단은 사용자의 몫

## 기술 스택

- Next.js 14 (App Router)
- Prisma + Supabase PostgreSQL
- next-intl (한국어/영어)
- Football-Data.org API (무료 플랜)
- Vercel 배포

---

**기억할 것**: PlayStat은 "분석 플랫폼"이지 "예측 플랫폼"이 아닙니다.
